# The Next Step is a full-stack career exploration platform that helps users research career paths and plan their future.

**Live demo:** Currently being prepared for its next public release. Contact me for access or a technical walkthrough.

## Why This Is Interesting

The Next Step brings career research into one focused platform instead of making users search across disconnected websites. It organizes career and user data to support exploration, saved careers, and more personalized guidance.

## Tech Stack

- React
- JavaScript
- React Router
- Node.js
- Express
- MongoDB
- REST API
- CSS

## Technical Highlights

- **Full-stack data flow:** Career and user information is managed across a React frontend, REST API, backend service, and persistent database.
- **Extensible data model:** Career and user data are structured separately, creating a foundation for profiles, saved careers, and personalized recommendations.
